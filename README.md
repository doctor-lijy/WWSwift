# WWSwift

WEEX iOS 项目的 Swift 组件库，以 CocoaPods 形式提供。

## 要求

- iOS 14.0+
- Swift 5.0+

## 安装

### CocoaPods（远程）

```ruby
pod 'WWSwift', :git => 'https://github.com/doctor-lijy/WWSwift.git', :tag => '0.1.0'
```

### CocoaPods（本地路径）

```ruby
pod 'WWSwift', :path => '../WWSwift'
```
